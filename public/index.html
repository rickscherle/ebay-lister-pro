ListAI uses Claude to analyze your photos and generate listings. You need your own Anthropic API key. Each listing costs roughly $0.01–$0.03. New accounts get $5 in free credit.

  1. Go to console.anthropic.com and create an account
  2. Add a payment method under Settings → Billing
  3. Create a key under Settings → API Keys (starts with sk-ant-)
  4. Paste it below
